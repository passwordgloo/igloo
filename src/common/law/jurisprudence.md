## 法的概述

::: details 法律定义

国家指定认可，国家强制力保障实施（最后保障），反映物质决定的统治阶级意志权利与义务为基础的行为规范体系

亚里士多德是西方第一个阐述法治学说和行政概念的人

:::

![](/common/法律与国家.png)

### 法的价值

法按照制定和实施法律主体可分为国内法和国际法

法的价值在于自由（核心）、秩序、正义，是社会主义法治理念价值追求，也是中国特色社会主义内在要求

### 法的特性

| 特性       | 解释                         |
| ---------- | ---------------------------- |
| 稳定性     | 相对稳定性，法是一成不变的❌  |
| 可诉性     | 用于诉讼                     |
| 程序性     | 程序正当                     |
| 普遍性     | 法律面前人人平等；立法不平等 |
| 强制性     | 必须遵守                     |
| 规范性     | 制约行为                     |
| 不可违抗性 | 法律权威                     |
| 国家一致性 |                              |
